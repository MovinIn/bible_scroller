# Word study data attribution

## BSB display groupings (`display/`)

Berean Standard Bible word-level Strong's groupings from
[BSB-publishing/bsb-data-output](https://github.com/BSB-publishing/bsb-data-output)
(`base/display/`). License: **CC0 (Public Domain)**.

Fetch with `scroller/server/scripts/fetch_word_study_data.ps1`.

([openscriptures/strongs](https://github.com/openscriptures/strongs)).
